A {\itshape Buffer\+Allocator} is an {\itshape Allocator} which operates on a contiguous block of buffer memory, usually pre-\/allocated by an upstream memory source. {\itshape Buffer\+Allocator}s are typically used in scenarios where memory allocation and deallocation need to be fast and predictable, such as in real-\/time systems or high-\/performance applications. Optionally, a {\itshape Buffer\+Allocator} may provide thread-\/safe guarantees for concurrent allocator operations, in which case it is a {\itshape Synchronized\+Buffer\+Allocator}. If it further allows construction and assignment between its non-\/synchronized counterpart and itself, it is a {\itshape Convertible\+Buffer\+Allocator}.\hypertarget{md_docs_named_reqs_BufferAllocator_autotoc_md1}{}\doxysection{Requirements}\label{md_docs_named_reqs_BufferAllocator_autotoc_md1}
