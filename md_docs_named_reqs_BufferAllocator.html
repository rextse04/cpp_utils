Named Requirement: BufferAllocator

A BufferAllocator is an Allocator which operates on a contiguous block of buffer memory, usually pre-allocated by an upstream memory source. BufferAllocators are typically used in scenarios where memory allocation and deallocation need to be fast and predictable, such as in real-time systems or high-performance applications. Optionally, a BufferAllocator may provide thread-safe guarantees for concurrent allocator operations, in which case it is a SynchronizedBufferAllocator. If it further allows construction and assignment between its non-synchronized counterpart and itself, it is a ConvertibleBufferAllocator.

Requirements

A class type A meets the BufferAllocator named requirement if it satisfies Allocator and all requirements below.
