Detailed Description

template<type_qualifiers Q>
class utils::basic_obj_ptr< Q >

Thin wrapper around a void pointer that acts like a this pointer, which allows the dyn_method template to specialize for non-static dynamic methods. Must be used in non-static dynamic methods and as the first parameter.
