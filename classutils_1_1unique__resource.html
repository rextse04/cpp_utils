Detailed Description

template<typename T, bool Sync = false, typename Semaphore = std::counting_semaphore<>, typename SharedMutex = std::shared_mutex>
class utils::unique_resource< T, Sync, Semaphore, SharedMutex >

This class provides a manager of a user-provided resource that is capable of limiting and synchronizing access to it. Access to the underlying resource is done via "permits" that are issued by the class.

Template Parameters
Syncadds an internal mutex such that modifications via permit is synchronized (this should not be set to true if T is inherently synchronized, such as atomics)
Semaphorea type that follows the interface of std::counting_semaphore
SharedMutexa type that satisfies SharedMutex (can be any type if Sync is false)
Remarks
In the context of this class, read-only access is synonymous with const reference, while read/write access is synonymous with non-const reference (to the underlying resource).
